# Tools & Equipment

List what each DIY task needs, with honest economics.

## How to reason
- Per task: tool, purpose, rent-or-buy note (rent a tile saw unless they tile often).
- Fetch current tool prices/availability where useful; suggest economical alternative methods that achieve the
  same result.
- Keep it to the non-Tier-1 tasks the family is actually doing.

## Output
`tools_required[]` (tool, purpose, rent_or_buy_note).
